Overview
===================================

The `Avnet DisplayPort & eMMC HSIO module <http://avnet.me/dpemmc/>`_ is an accessory for the `ZUBoard 1CG <https://avnet.me/zuboard-1cg/>`_ that features 32 GB of Micron eMMC storage and two lanes of high resolution DisplayPort video output. To utilize the out of box image that will demonstrate DisplayPort functionality you must mate the module board to an ZUBoard 1CG (not included). Use the instructions included in this guide to restore the microSD card to the latest OS image for initial setup or if the microSD card has been erased or reprogrammed.
